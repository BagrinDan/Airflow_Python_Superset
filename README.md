# Airflow_Python_Superset

An end-to-end ETL pipeline that demonstrates a modern data engineering workflow.

The project uses **Apache Airflow** for workflow orchestration and scheduling, **Python** for data extraction, cleaning, transformation, and loading, **Azure Data Lake Storage Gen2** as the data lake for storing raw and processed datasets, **PostgreSQL** as the analytical data warehouse, and **Apache Superset** for interactive dashboards and data visualization.

The pipeline automatically ingests data, applies transformation and validation processes, stores intermediate and final datasets in the appropriate storage layers, and exposes business insights through interactive dashboards.

Example dashboards can be found in the **pictures** folder.
